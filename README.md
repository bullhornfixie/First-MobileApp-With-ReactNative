# IndoorJungle 

This was a personal project to build my first mobile app with React Native + Expo. 

IndoorJungle is a mobile app which helps users look after their indoor plants by providing information about the correct amount of water and light for their selected plant. 

## User Stories 

```
As a plant enthusiast 
With variety of indoor plants 
I want to make sure my indoor plants do not die 
```
